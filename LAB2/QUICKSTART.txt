🐛 TROUBLESHOOTING:
━━━━━━━━━━━━━━

Q: "omp.h: No such file or directory"
A: Install OpenMP development files:
   Ubuntu/Debian: sudo apt-get install libomp-dev
   macOS:         brew install libomp

Q: Programs run but no speedup observed
A: Check:
   1. OMP_NUM_THREADS is being set correctly
   2. CPU throttling isn't active (check scaling_governor)
   3. Background processes aren't consuming CPU
   4. Compile with -O3 optimization

Q: Results seem inconsistent
A: Try:
   1. Close background applications
   2. Run multiple times and average results
   3. Disable CPU frequency scaling
   4. Pin threads with taskset
